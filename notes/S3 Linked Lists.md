<h3 id="big-o">Big O</h3>
<ul>
<li>adding end to the end is O(1)</li>
<li>removing end is O(n)</li>
<li>adding at head is O(1)</li>
<li>removing at head is O(1)</li>
<li>adding in between is O(n)</li>
<li>removing in between is O(n)</li>
<li>finding a value by index or value both re O(n)</li>
</ul>
